## Layout

We use the **Content Grid** system to manage responsive widths.

| Class | Width | Usage |
| :--- | :--- | :--- |
| `content-grid` | 100% | Container with horizontal padding |
| `content` | 900px | Standard readable text width |
| `breakout` | 1200px | Image or wide table breakout |
| `full-width` | 100% | Edge-to-edge content |
