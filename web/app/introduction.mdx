# Introduction

Welcome to **CRAN/E** (The Comprehensive R Archive Network, Enhanced). This is our component library and design system documentation.

## At a Glance

- **Fast Search**: Lightning-fast indexing and retrieval for tens of thousands of R packages.
- **Modern PWA**: Built with React Router v7 and Tailwind CSS v4 for a smooth, app-like experience.

## Key Technologies

- **React Router v7**: Modern routing and data loading.
- **Tailwind CSS v4**: Utility-first styling with native CSS variables.
- **Radix UI**: Unstyled, accessible UI primitives.
- **MCP**: Model Context Protocol for AI integrations.

---

## Component Architecture

Our components are located in `app/modules`. They are designed to be:
1. **Framework-agnostic** where possible.
2. **Accessible** out of the box.
3. **Themable** with dark mode support.
